#Diabeties-Detection-Using-ML
Diabetes Detection Using Machine Learning

##Description
Machine Learning-based application to predict whether a person is diabetic using health-related parameters and the K-Nearest Neighbors (KNN) algorithm.

###Features
Predicts diabetes based on patient health data.
Uses KNN classification algorithm.
Data preprocessing and feature scaling.
Accuracy evaluation using train-test split.
User-friendly prediction interface.
Displays prediction results instantly.

####Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Jupyter Notebook
Dataset Features
Pregnancies
Glucose Level
Blood Pressure
Skin Thickness
Insulin
BMI
Diabetes Pedigree Function
Age
